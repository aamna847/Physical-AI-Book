# Physical AI Book

Welcome to the Physical AI Book repository! This is a comprehensive guide to building humanoid robots with artificial intelligence.

## About

This book covers the complete journey of building humanoid robots, from fundamental concepts to advanced implementations. It explores topics like:

- Sensors and perception
- ROS2 architecture and core concepts
- Simulation environments with Gazebo and Unity
- NVIDIA Isaac Sim SDK
- Vision-language-action models
- Humanoid kinematics and locomotion
- Hardware requirements and platforms

## Running the Documentation

This documentation is built with Docusaurus:

```bash
# Install dependencies
npm install

# Start the development server
npm start
```
